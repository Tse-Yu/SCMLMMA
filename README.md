# 2018 NCTS Short Course on Scientific Computing and Machine Learning on Multi- and Manycore Architectures 

This folder contains 7 exercises in 2018 NCTS Short Course on Scientific Computing and Machine Learning on Multi- and Manycore Architectures .

See <a href="https://sites.google.com/site/school4scicomp/2018-a-anzt" target="_blank">here</a> for course detail.

In this short course, there are 7 exercieses. These exercise would be briefly introduced below.

* Vector-vector, matrix-vector, and matrix-matrix multiplication.
* LU decomposition with and without pivoting.
* Block LU decomposition and its OpenMP, OpenMP task/task dependence version.
* Format transform between Coordinate Format (COO) to Compressed Sparse Row Format (CSR) of sparse matrix.
* SpMV on GPU.
* Bandwidth Test of Message Passing Interface (MPI).
* Tutorial of cuDNN model from <a href="http://www.goldsborough.me/cuda/ml/cudnn/c++/2017/10/01/14-37-23-convolutions_with_cudnn/" target="_blank">Peter Goldsboroughâ€™s example</a>.
